# RSM Plumbing Website (Customized Version)

This project is a customized and fully redesigned version of the original **Wib Next.js template** downloaded from ThemeWagon. All design improvements, structural changes, new components, and full implementation were done by **Baraka Juma Mashimbe**.

## 🔧 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Export static files
npm run export
```
